mvc的架构

sql语句结构化查询语言

drop database if exists fang 删除

create database fang default charset utf8
collate utf8_bin 创建数据库

use fang

创建虚拟环境

python -m venv +虚拟环境名

pip freeze 查看依赖库

输出重定向

pip freeze > requirements.txt
把依赖库放到txt文件里面

安装
pip install -r requirements.txt

pip check 检查安装依赖库

创建django 项目

django-admin startproject fang .

dir 查看

创建应用

python manage.py startapp common

修改配置文件

python manage.py inspectdb > common/models.py 检查数据库把检查的结果输出到common/models.py 中

init 在这个文件中写上下面几句

import pymysql

pymysql.install_as_MySQLdb()
